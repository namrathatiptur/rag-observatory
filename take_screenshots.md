# Dashboard Screenshots Guide

Since I cannot directly capture browser screenshots, here's how to take them:

## Using macOS Screenshot Tool

1. **Open the dashboard**: http://localhost:8501
2. **Take screenshots**:
   - Press `Cmd + Shift + 4` for selection screenshot
   - Press `Cmd + Shift + 3` for full screen
   - Press `Cmd + Shift + 4` then `Space` for window screenshot

## Recommended Screenshots

1. **Main Dashboard View** - Home page with configuration
2. **Query Documents Mode** - Upload interface
3. **Live Query Results** - After querying with metrics
4. **Metrics Analysis Tab** - Charts and visualizations
5. **Failure Analysis Tab** - Failure detection view
6. **Query Inspector Tab** - Query history
